# Folder Assets Gambar Linimasa

Folder ini digunakan untuk menyimpan seluruh gambar pendukung aplikasi (seperti logo, background, dsb) yang akan dimuat secara publik.

- `/characters` : Simpan gambar karakter, maskot, avatar, atau ilustrasi tokoh di sini.
- `/logos` : Simpan gambar logo utama aplikasi atau ikon kustom di sini.
